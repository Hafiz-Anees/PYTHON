## 1-write a program to get day , month and year from user and print in this format 29-09-2023.

- Solution

```python
day = input("Enter day (DD): ")
month = input("Enter month (MM): ")
year = input("Enter year (YYYY): ")

print(day, "-", month, "-", year)
```
